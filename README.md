# 📑 Dev & Data Resume — Natan G. Almeida

Este repositório contém o código-fonte do meu currículo web e portfólio profissional. O projeto foi desenvolvido com uma identidade visual moderna, minimalista e com uma estética *dark mode/cyberpunk*, projetada para destacar minhas competências em **Desenvolvimento Web, Infraestrutura e Análise de Dados**.

---

## 🚀 Tecnologias Utilizadas

O projeto foi construído prezando pela leveza, semântica e performance estrutural:

* **HTML5:** Estruturação semântica e acessível encontrada no arquivo `index.html`.
* **CSS3 (Custom Properties):** Estilização customizada contida no arquivo `style.css` utilizando variáveis globais (`:root`), efeitos de gradiente de texto, transições suaves, efeitos de *glow* neon e responsividade para dispositivos móveis.
* **Fontes:** Combinação de tipografia clássica de sistema (`Segoe UI`, `Roboto`) com elementos *monoespaçados* (`Courier New`) para reforçar a estética tech.

---

## 🎨 Design & Funcionalidades

* **Cyber-Dark Theme:** Paleta de cores inspirada no ecossistema de desenvolvimento (tons de grafite, azul e ciano neon).
* **Organização em Blocos:** Layout limpo e escaneável dividindo Perfil, Experiência, Formação, Projetos e Certificações.
* **Interatividade Baseada em CSS:** Efeitos visuais dinâmicos de *hover* nos cards de experiência, links de projetos e *tags* de tecnologia.
* **Design Responsivo:** Adaptável tanto para visualização em desktops quanto para telas de smartphones.
